## v1.0.0 - Initial Release

Initial public release of the PowerShell Endpoint Automation Toolkit.

Included scripts:

- Audit-OrphanedUserProfiles.ps1
- Collect-SystemDiagnostics.ps1
- Scan-EndpointHealth.ps1
- Set-LockTimeout.ps1
- Test-DeploymentReadiness.ps1

This release provides example PowerShell scripts demonstrating practical
automation and diagnostics techniques for Windows endpoints.

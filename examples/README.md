# Example Scripts

This repository provides example PowerShell scripts that demonstrate practical automation techniques commonly used in Windows endpoint administration.

This directory contains example PowerShell scripts demonstrating common endpoint automation and diagnostic tasks.

These scripts are intentionally generic and sanitized so they can be safely shared publicly.

## Scripts Included

`Audit-OrphanedUserProfiles.ps1`
Scans the Windows profile registry and reports profiles that may be orphaned or in a failed state.

`Collect-SystemDiagnostics.ps1`
Collects troubleshooting information useful for diagnosing endpoint problems.

`Scan-EndpointHealth.ps1`
Captures a quick health snapshot of a Windows endpoint.

`Set-LockTimeout.ps1`
Enforces an inactivity lock timeout by configuring the Windows registry policy.

  `Test-DeploymentReadiness.ps1`  
Checks whether a system is ready for software deployment.

These examples are intended to demonstrate practical PowerShell automation patterns used in real-world Windows environments.

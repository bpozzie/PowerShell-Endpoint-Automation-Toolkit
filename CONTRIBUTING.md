# Contributing

Thank you for your interest in contributing.

This repository contains sanitized PowerShell examples intended to
Demonstrate practical endpoint automation and diagnostics techniques.

## Guidelines

When contributing or modifying scripts:

- Avoid including organization-specific infrastructure
- Avoid hardcoding domain names or internal network paths
- Use clear and consistent PowerShell formatting
- Include comment-based help when possible

## Purpose of the Repository

The goal of this repository is to provide clean examples of PowerShell
automation patterns used in real-world Windows environments.

Scripts should remain:

- generic
- portable
- safe for public sharing
